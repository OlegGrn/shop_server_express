Для работы установить
npm i nodemailer

 - делаем уникальную строку и сохраняем ее в базе у пользователя.
 - к этой уникальной строке добавляем адрес для активации - это наш эндПоинт, и отправляем
 письмо пользователю с этой ссылкой.
 Когда пользователь нажмет на ссылку (а это наш эндПоинт), мы получим из запроса хвостик (уникальную
 строку), по ней найдем в базе юзера и обновим его данные что почта подтверждена.
 Затем перенаправим пользователя на главную страницу приложения.

 Для настройки nodemailer в переменные окружения вынесем все основные данные
 url нашего эндПоинта, Хост почты, порт почты, саму почту, пароль к ней, и куда потом перенаправить пользователя

В контролере в блок регистрации передаём функцию из маил сервис для вызова функции по
отправке сообщения на почту юзеру

В контролере '/activate/:link':
1 получаем уникальную строку из переменной link
2 по этой ссылке ищем пользователя в базе
3 изменяем поле активации почты в положение true
